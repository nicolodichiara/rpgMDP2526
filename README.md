# rpgMDP2526 - Fishing RPG

A JavaFX-based role-playing game centered around the art of fishing. Players explore diverse maps, level up, manage their inventory, and strive to catch every fish species available.

### Gameplay Overview

* **Exploration:** Free movement across grid-based maps.
* **Combat/Fishing:** Engaging turn-based battles between the player and the fish once an encounter is triggered.

---

## Getting Started

### Prerequisites

Make sure you have the following installed on your system:

* **Java 25 (LTS)**
* **Gradle**

### Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/nicolodichiara/rpgMDP2526.git

```


2. **Navigate to the project directory:**
```bash
cd rpgMDP2526

```



### Building the Project

To build the project using Gradle, run:

```bash
./gradlew build

```

### Running the Application

To launch the game, execute:

```bash
./gradlew run

```
